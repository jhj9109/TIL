삽입 정렬

> 연결리스트일때 삽입정렬이 효율적

- S : 정렬된 앞부분의 원소들
- U: 정렬되지 않은 나머지 원소들
- U의 원소를 하나씩 S에 삽입정렬
- 시간 복잡도 O(n^2)

```python
arr = [69, 10, 30, 2, 16, 8, 31, 22]
N = len(arr)
for i in range(1, N):
    tmp = arr[i]
    print(arr[:i], arr[i:])
    j = i - 1
    while j >= 0 and arr[j] > tmp:
        arr[j+1] = arr[j]
        j -= 1
    arr[j+1] = tmp
print(arr)
```
